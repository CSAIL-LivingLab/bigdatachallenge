Dataset Descriptions

Taxi

The taxi data contains a set of "pickupX.csv" and "dropoffX.csv" files, where X is the month of each dataset. The files contain time, address, and locations of every pickup and dropoff in central boston.

MBTA

This contains AFC and OFC files.

AFC files contain information about bus ridership (every time a rider pays the driver). The station information is inferred so it may not be accurate.

OFC files contain swipe information for the T stations and are accurate.

The Stops.rpt file contains station metadata information.

Weather

This contains a JSON encoded dump of Weather Underground's historical weather data in boston. The data is encoded as a list of JSON objects, where each object is the hourly data for one day. See the API documentation for more details.

Events

This contains the location, time, and description information about all Boston events.
